Hier baue ich eine Wetter-App. Es ist mein erstes grösseres Projekt nach der Witze-App.
Es soll ein Inputfeld bekommen und dort soll man einen beliebigen Ort eintragen können. Über die API werden 
die Orte gesucht und die verschiedenen Parameter angezeigt. Wenn ich einen neuen Ort suchen möchte, gehe ich einfach mit dem Pfeil zurück. Möchte ich
jedoch den gewünschten Ort zu meinen Favoriten hinzufügen ist dies möglich, sowie auch diesen Ort wieder zu löschen.
